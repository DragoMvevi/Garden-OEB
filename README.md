# Garden OEB

This is a static, multi-file website generated by the Site Generator.

## Structure
- `index.html` and other page HTML files at the root
- `css/style.css` — all styles
- `assets/` — folder reserved for your custom images (replace remote Unsplash URLs as you like)
- `robots.txt`, `sitemap.xml` — basic SEO files
- `_redirects` — Cloudflare Pages SPA-style fallback (safe to keep)

## Deploy to Cloudflare Pages (free, automatic)

### Option A — Drag & drop (60 seconds, no GitHub needed)
1. Go to https://dash.cloudflare.com/ → **Workers & Pages** → **Create** → **Pages** → **Upload assets**.
2. Give it a project name and drag this entire folder (or the .zip) in.
3. Click **Deploy site**. You will get a free `*.pages.dev` URL immediately.

### Option B — Connect GitHub for auto-deploy on every push
1. Create a new GitHub repository and upload the contents of this folder to it.
2. In Cloudflare → **Workers & Pages** → **Create** → **Pages** → **Connect to Git**.
3. Pick the repo. Leave **Framework preset = None**, **Build command = (empty)**, **Build output directory = /** (or `./`).
4. Click **Save and Deploy**. Every `git push` redeploys automatically.

### Custom domain
In the Pages project → **Custom domains** → add yours and follow the DNS instructions (Cloudflare auto-provisions HTTPS).
